====
busy
====

tkinterで作成したPythonエディタ

Requirement
===========
:Python: 3.5以上


Install
===========
1. pipでインストールする ::

    pip install -U https://github.com/naritotakizawa/busy/archive/master.tar.gz


2. うごかす ::

    busy


File Shortcut Key
===========
:Ctrl+N: 新規ファイル
:Ctrl+O: ファイルを開く
:Ctrl+D: 現在のタブを閉じる
:Ctrl+S: ファイルを保存・更新する


Editor Shortcut Key
===========
:Enter: 改行する。適切にインデントを行って改行します。
:Tab: インデントする。選択してれば範囲インデント
:Ctrl+B: 範囲を逆インデントする
:Ctrl+A: テキストを全て選択
:Ctrl+L: テキストを全てハイライトし直す
:Ctrl+C: コピー
:Ctrl+X: 切り取り
:Ctrl+V: 貼り付け


Tree Shortcut Key
===========
:F4: ツリーのルートディレクトリ変更
:F5: ディレクトリツリーの更新
