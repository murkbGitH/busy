====
busy
====

tkinterで作成したPythonエディタ

Requirement
===========
:Python: 3.5以上


Install
===========
1. pipでインストールする ::

    pip install -U https://github.com/naritotakizawa/busy/archive/master.tar.gz


2. うごかす ::

    busy


Usage
===========
:Ctrl+N: 新規ファイル
:Ctrl+O: ファイルを開く
:Ctrl+D: 現在のタブを閉じる
:Ctrl+S: ファイルを保存・更新する
